# 🎨 Emotion Canvas — SillyTavern Extension

Расширение, которое **в реальном времени визуализирует эмоциональное состояние персонажа** через динамические частицы, цветовую атмосферу и живой градиентный фон.

---

## 📦 Установка

1. Скопируй папку `emotion-canvas` в:
   ```
   SillyTavern/public/extensions/third-party/emotion-canvas
   ```
2. Запусти SillyTavern
3. Зайди в **Extensions → Manage Extensions** и включи **Emotion Canvas**
4. Нажми кнопку 🎨 в правом нижнем углу чата

---

## ✨ Возможности

| Функция | Описание |
|---|---|
| **Авто-определение эмоций** | Анализирует текст сообщений и определяет эмоцию |
| **Частицы** | Уникальные анимации для каждой эмоции (искры, дождь, сердечки...) |
| **Градиентная атмосфера** | Фон меняет цвет в зависимости от настроения |
| **История эмоций** | Мини-график изменения настроения за диалог |
| **Mood Lock** | Зафиксировать текущую атмосферу |
| **Cinematic Mode** | Полупрозрачный чат поверх атмосферного фона |

---

## 🎭 Поддерживаемые эмоции

- ✨ **Радость** — золотые искры
- 💕 **Любовь** — розовые сердечки
- 🌧️ **Грусть** — синий дождь
- 🔥 **Гнев** — огненные угли
- 🌑 **Страх** — фиолетовые тени
- 🌊 **Удивление** — голубые пузыри
- 🌿 **Спокойствие** — тёмный нейтральный фон

## 📝 Как работает детекция

Расширение анализирует **текст последнего сообщения** по словарю ключевых слов для каждой эмоции на русском и английском языках. Интенсивность вычисляется по количеству совпадений. Никаких дополнительных API-запросов — всё работает мгновенно и оффлайн.
